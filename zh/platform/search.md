# 搜索 GitBook

你可以使用 [我们强大的搜索工具](https://www.gitbook.com/search) 从 GitBook 数以千计的书籍中查找你需要的书籍.

当搜索 GitBook 的时候, 你可以构建匹配数字和单词的查询.


### 文本搜索

默认情况下, GitBook 会搜索与查询关键字相关的书籍. 例如 `javascript angular` 将会返回所有的包含单词 "javascript" 和 "angular" 的书籍.

### 排除包含某个结果的单词

你还可以缩小你的搜索结果通过使用 `NOT` 语法排除一些单词. 搜索 `Hello` 返回大量的 "Hello World" 项目, 但是改变你的搜索用 `hello NOT world` 返回更少的结果.

### 查询特定的字段值

你可以通过书籍包含的特定字段值过滤查询结果. 例如: [license:apache-2](https://www.gitbook.com/search?q=license%3A%22apache-2%22) 返回使用 Apache 2 许可的图书列表.

### 值少/大于另一个值的查询

你可以使用 `>` 或者 `>=` 分别表示 "大于" 和 "大于或等于". 例如, 下面的搜索是等价的: `cats stars:">10"` and `cats stars:">= 11"`

你可以使用 `<` 或者 `<=` 分别表示 "小于" 和 "小于或等于".
